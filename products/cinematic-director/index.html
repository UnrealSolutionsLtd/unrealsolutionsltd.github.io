FAQ

How is this different from Epic's Camera Animation Toolset?

CAT is shot-level camera authoring inside Sequencer. The AI Cinematic Director ingests event streams from outside Unreal, produces multi-hour coherent output across an entire session, and generates event-aligned text overlays. External ingestion, multi-hour coherence, and overlays are outside CAT's scope.

What event stream protocols are supported?

The ingestion layer is protocol-agnostic. Adapters for common virtual production, defense simulation, and robotics event formats are planned. New adapters are a defined extension point.

What Unreal Engine versions are supported?

Unreal Engine 5.4+. Will integrate with the Runtime Video Recorder for real-time capture.
