## System Requirements

### Unreal Engine Version

- **Minimum:** Unreal Engine 5.0
- **Recommended:** Unreal Engine 5.3 or higher
- **Supported:** All major UE5 versions

### Platform Requirements

#### Windows
- OS: Windows 10/11 (64-bit)
- GPU: DirectX 11+ compatible
- RAM: 8 GB minimum, 16 GB recommended
- For HW acceleration: NVIDIA Kepler+ (GTX 600), AMD GCN 1.0+, Intel HD 4000+

#### macOS
- OS: macOS 10.15 (Catalina) or higher
- Apple Silicon and Intel Macs supported
- RAM: 8 GB minimum, 16 GB recommended

#### Linux
- Ubuntu 20.04+ or equivalent
- RAM: 8 GB minimum, 16 GB recommended
- Note: Software encoding only (OpenH264)

#### Android
- Android 5.0 (API 21) or higher
- RAM: 3 GB minimum
- Hardware H.264 encoding support (most devices 2015+)

#### Oculus/Meta Quest
- Quest 1, Quest 2, Quest 3, Quest Pro
- Android-based platform requirements apply
